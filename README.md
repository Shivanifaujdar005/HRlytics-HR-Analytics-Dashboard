# HRlytics - HR Analytics Dashboard

## Overview

HRlytics is a Django-based HR Analytics Dashboard developed to streamline HR operations and provide actionable workforce insights. The platform enables employee attendance management, secure authentication, and centralized HR data monitoring through an intuitive web interface.

## Features

- Employee attendance tracking and management
- Secure user authentication and login system
- Interactive HR analytics dashboard
- Workforce data monitoring and reporting
- File upload and management functionality
- Responsive and user-friendly interface

## Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- JavaScript

## Project Structure

- Dashboard module for analytics and reporting
- Attendance management system
- User authentication and authorization
- Media and file management support
- Database-driven HR records management
